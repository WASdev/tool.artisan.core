Spring Boot with WebSocket

This project demonstrate how to accessing relational data with Spring technology and run on IBM WebSphere Liberty Server

Learn more about IBM WebSphere Liberty, visit https://developer.ibm.com/wasdev/

Inside the application project there is a application.springboot.websocket package containing four classes:

  • Application: The entry point for the SpringBoot application
  • MvcConfig: Map each uri to static html resources
  • SystemWebSocketHandler: Creating a WebSocket server by implementing WebSocketHandler
  • WebSocketConfig: The WebSocket config class to map the WebSocket handler at a specific URL
  • View how application work at:/springbootwebsocket

Inside the wlpcfg project there is the it.springboot.security.HelloControllerTest that will test the Spring WebSocket endpoint to ensure it is working.
