Spring Boot with Spring MVC and REST controllers

Inside the project there is a application.springboot.web package containing these classes:

  • SpringBootLibertyApplication: The entry point for the SpringBoot application.
  • LibertyMVController: A Spring MVC endpoint which you can access at /springbootmvc
  • LibertyRESTController: A Spring REST endpoint which you can access at /springbootrest

There is also a test class named it.springboot.web.HelloControllerTest that will test the Spring MVC endpoint to ensure it is working.
